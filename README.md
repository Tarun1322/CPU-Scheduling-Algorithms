Intelligent CPU Scheduling Simulator
Overview
An interactive web-based simulator that visualizes how different CPU scheduling algorithms work. The project helps students understand process scheduling with graphical Gantt chart representation and performance metrics.

Features
Supports multiple algorithms: FCFS, SJF, SRTF, Round Robin, Priority (Preemptive & Non-Preemptive)
Real-time Gantt chart visualization
Calculates Completion Time, Turnaround Time, Waiting Time, and Response Time
Simple and user-friendly interface with dark mode support
Live Demo
https://cpu-scheduling-algorithm.onrender.com

Tech Stack
HTML
CSS
JavaScript
Usage
Select a scheduling algorithm.
Enter process details (arrival time, burst time, priority if needed).
Click Calculate to generate results and visualize the scheduling process.
